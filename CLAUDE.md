# Project Overview

Personal portfolio website for Vladyslav Kovalchuk. The #1 goal is to sell the author as a senior frontend React developer at the highest possible rate. Every page, every article, every detail should maximize perceived value and justify premium pricing.

## What the site does

- Presents professional profile, tech stack, and work philosophy
- Showcases case studies (real projects with technical depth)
- Hosts a blog with opinionated engineering articles on React, TypeScript, and frontend architecture
- Provides a hire/contact page for potential employers and clients

## Tech stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Key principles

- Every page should reinforce senior-level expertise: thoughtful architecture, real-world tradeoffs, opinionated technical writing
- Blog posts should be practical, concise, and demonstrate deep understanding — not tutorials for beginners
- Design is minimal and clean — the content sells, not the decoration
